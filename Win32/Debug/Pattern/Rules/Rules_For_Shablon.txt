1. любая "строка для вставки" должна находиться между двумя долларами ($ *текст* $). 
2. любая "строка для вставки" должна находиться на одной строчке и не переносится.
3. У каждой "строки для вставки" есть свой приоритет, если до этого не было найдено ничего что могло определить что надо вставить в эту строку для вставки то она запрашивается у пользователя.
4. Нельзя делать вложенные друг в друга "строки для вставки"
5. Любой текст между 2 долларами будет проверяться на "строку для вставки"
6. Регистр букв в "строке для вставки" не важен.
7. например надо вывести 2 списка групп, программа для определения какая группа будет сначала искать наиболее ранее заранее определённую группу, если не найдёт то будет смотреть что говорили потом.
 
5. Поддерживаемые "строки для вставки":
$ФИО$   //ФИО ученика в родительном падеже
$деканД$ //декан в дательном падеже
$допФИО$ //любое ФИО, всегда спрашивается, так же стоит использовать для вставки любой строки в любое место  
$группаФИО$ //ФИО всей группы, каждое ФИО с новой строки, пронумерованные
$группа$   //номер группы 
$семестр$  
$факульт$  // факультет
$спец$     // специальность  
-------------$спецФакультета$  //все специальности, только по указанному факультету
$предмет$         //учебный предмет  
$ПФИО$    //ФИО преподавателя (по указанному предмету или вводится) должно быть с должностью?
$дата$    //дата создания документа
$мояДата$ //ввести дату мероприятия
$формаОбучения$
$пересдач$  //число от 0 до 100
$телефон$   //номер телефона само в проге +375 дальше принимаем строку
$печать$    //гербовая, обычная
$местоПредъявления$  //для кого справка
$видЗанятий$        //лабы, курс работа, зачёт
$$

справки формируются только по имени, тип печати
ведомость -- предмет, видЗанятий, фио

Обязательно к вводу ФИО, $предмет$ $пересдач$ $телефон$ $печать$ $местоПредъявления$ $видЗанятий$** $мояДата$
порядок: обязательные, то что из них получается,
 
ввод 2 необходимости, если обязательных нет    группа, $ПФИО$
